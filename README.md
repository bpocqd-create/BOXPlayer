# BOX Player - Android

## كيف تبني الـ APK (5 خطوات بس)

### 1. حمّل Android Studio
https://developer.android.com/studio (مجاني)

### 2. افتح المشروع
- افتح Android Studio
- اختر "Open an existing project"
- اختر مجلد BOXPlayer هذا

### 3. انتظر التحميل
Android Studio حيحمل المكتبات تلقائياً (يحتاج إنترنت، دقيقتين تقريباً)

### 4. بناء الـ APK
- من القائمة العلوية: Build → Build Bundle(s) / APK(s) → Build APK(s)
- انتظر دقيقة

### 5. موقع الـ APK
```
BOXPlayer/app/build/outputs/apk/debug/app-debug.apk
```
انقله للجوال وثبّته ✅

## مميزات التطبيق

- 📁 إضافة مجلدات مسلسلات من التخزين الداخلي
- 🎬 تشغيل فيديو بـ ExoPlayer (يدعم MP4, MKV, AVI, ...)
- 📺 قائمة مواسم + حلقات جانبية
- 💾 حفظ موضع كل حلقة تلقائياً
- 📄 ترجمة تلقائية (SRT + ASS/SSA)
- ⏱ تعديل تأخير الترجمة (-5s → +5s)
- 🔤 تغيير حجم الترجمة
- ⏭ تشغيل الحلقة التالية تلقائياً
- 👆 Double tap للتقديم/الترجيع
- 🌙 تصميم داكن بالبرتقالي

## متطلبات
- Android 7.0 (API 24) وما فوق
